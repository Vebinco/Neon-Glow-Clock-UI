# Neon Digital Clock

A futuristic Neon Digital Clock UI with glowing effects, built with HTML, CSS, and JavaScript.

## Features

- **Futuristic UI:** A sleek, high-contrast design inspired by sci-fi dashboards.
- **Neon Glow Effect:** Bold typography with a vivid neon glow and a subtle gradient.
- **Pulsing Animation:** A soft pulsing animation gives the digits a "breathing" effect.
- **Theme Switcher:** Toggle between three different neon color themes (blue, pink, purple).
- **Responsive Design:** The clock is fully responsive and looks great on all screen sizes.
- **Fade-in Animation:** A smooth fade-in animation when the clock loads.
